## Cách sử dụng

1. Tạo CSDL cấu hình lại file _connect.php
2. Import CSDL ở file databases/shopgame.sql vào CSDL vừa tạo

## Các chức năng chính

1. Đăng ký
2. Đăng nhập
3. Trang profile (/profile.php?id=<user_id>)
4. Trang quản trị viên (/admin/)
>
> 5. Updating....
